npm node.js package installer
=============================

DON'T INSTALL FROM APT-GET!!!
sudo apt-get install npm

DO THIS INSTEAD:
git clone git://github.com/isaacs/npm.git
cd npm/scripts
chmod +x install.sh
sudo ./install.sh
