npm node.js package installer
=============================

DON'T INSTALL FROM APT-GET!!!
sudo apt-get install npm

DO THIS INSTEAD:
git clone git://github.com/isaacs/npm.git
cd npm/scripts
chmod +x install.sh
sudo ./install.sh


installing command line grunt
================
sudo npm install -g grunt-cli

This is a system wide install (-g for global)
it alows you to type "grunt" in command line

install local project dependancies
===================================
in firesafe git root:

npm install

will read the package.json and install our project specific stuff (e.g. firebase API)
we will add additional dependancies over time
